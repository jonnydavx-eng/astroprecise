# raw/ — drop sources here

This is your inbox. Put anything you want to remember in here:

- Articles (paste as `.md` or `.txt`)
- Transcripts (YouTube, podcasts, meetings)
- PDFs
- Your own rough notes and thoughts

Then tell Claude **`ingest raw/<filename>`** (or `ingest everything new in raw/`).

Claude treats this folder as **read-only** — it reads your sources but never
edits or deletes them, so the original is always preserved and every wiki note
can trace back to it.

Naming tip: prefix with a date for easy sorting, e.g.
`2026-06-22-article-title.md`.
