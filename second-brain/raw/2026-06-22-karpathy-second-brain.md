The idea that hit 16 million views: stop using AI to write code, use it to build
a second brain.

You point Claude Code at a folder, drop in any source — an article, a transcript,
a PDF — and Claude reads it, links it, and files it into a living wiki of
everything you know. It compounds like interest: the more you feed it, the
smarter it gets.

Here's the whole thing:

- Install Obsidian, create a vault, open it in Claude Code
- Paste Karpathy's wiki idea file and tell Claude to build it
- Claude makes three folders: raw for sources, wiki for its pages, a CLAUDE.md
  that runs it
- Drop any source into raw and say "ingest this"
- Ask questions across everything, forever

Five minutes to set up, and you never start from a blank chat again.
