# The planets

Astrology reads **ten bodies** — the Sun and Moon (the *luminaries*) plus eight
planets. Each represents a drive or function; *where* it falls (which
[[zodiac-sign]] and [[astrological-houses|house]]) colours how that drive
expresses.

## Detail

| Body | Symbol | Keywords |
|---|---|---|
| Sun | ☉ | Identity, Ego, Vitality |
| Moon | ☽ | Emotions, Intuition, Subconscious |
| Mercury | ☿ | Communication, Intellect, Travel |
| Venus | ♀ | Love, Beauty, Harmony |
| Mars | ♂ | Action, Desire, Energy |
| Jupiter | ♃ | Expansion, Luck, Wisdom |
| Saturn | ♄ | Discipline, Karma, Structure |
| Uranus | ⛢ | Innovation, Rebellion, Change |
| Neptune | ♆ | Dreams, Illusion, Spirituality |
| Pluto | ♇ | Transformation, Power, Rebirth |

Each [[zodiac-sign]] has a **ruling planet** (e.g. Mars rules Aries, Venus rules
Taurus). A planet can appear **retrograde** — apparent backward motion, read as a
turning-inward of that planet's themes.

## Connections

- Placed into [[zodiac-sign|signs]] and [[astrological-houses|houses]] within a
  [[birth-chart]].
- Form [[aspects]] with one another.
- Rulership links each planet to the sign(s) it governs (see [[zodiac-sign]]).
