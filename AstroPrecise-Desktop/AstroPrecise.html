Local Setup

Running a Local Preview

  1. Ensure Python 3 is installed — open a terminal and type python --version to verify.
  2. Windows: double-click launch.bat in this folder. It starts a local server and opens your browser automatically.
  3. Mac / Linux: open a terminal here and run ./launch.sh. The site will open at http://localhost:8420.
  4. Keep the terminal or command window open while you browse. Close it to stop the server.

The local preview serves the website/ folder from the repo root. All features including the offline service worker, live SWPC feeds, and ephemeris calculations work the same way as the live GitHub Pages site. No internet required once the page has loaded.
